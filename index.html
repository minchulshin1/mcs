PROJECT IDEAS

My project is developing the smart phone application called “handy job founder” which can mainly support university student to find their appropriate future job and obtain decent information. This will be made as team based project about 3 – 4 people who are in programming related field and it is expected to take approximately 1 month to finish this project.

There are approximately 700,000 graduated students who is seeking for their job appeared every year in Australia. it is very fatal that huge amount of students literally have no work to do and can’t get a job easily. but when we are trying to find job information from the internet which has vast data, it is very difficult to find as well as the recent article, many students get difficulties to make a right choice for their work and recognize what kind of job they really desire. What if there is application which can recommend the job for users through analyzing users’ interests and tendency? It will be much easier for people to approach perfect job. So that it is indeed vital giving a primary information regarding the job and workplace and recommend the appropriate job. This application might be one of the useful option for students to be aware of job selection.

This application will seek job by 2 stream, first one is finding jobs through own research, this is the vast database storage to make users to find important information such as wages, qualification requirements, skills requirement. Users can find relevant firm in chosen area by automatic GPS and set a location in the simple tool bar so that users can only find related information. But some people might have no idea what they are interested in or what they want to be then this is the case for user to take second stream. Before using this, users need to make a account to make their personal profile on this application. They need to take 3 representative tests job test, personality test, life style test and then based on these 3 tests results, the application make users’ resume automatically through analyzing what users interests are and what users want. After all of these steps, the application will recommend users appropriate job and user can consider qualifying condition. If user dislike, they can swipe left which means that they want to see other firm. If they like the current post, they can click the apply button then the application will send users’ resume automatically. This is the basic process of the application. Not only these functions, there is a user library function which has a lot of job information for example if they want to be programmer, what program they should know how to use or experiences requirements. Information are clearly given in this category so that users can obtain what they wanted to know. What the good thing of this application is that the application provides tremendous information about the firm that user is looking for and help to connect with firm and user 1 by 1. They can have a rough talks and user can get a great opportunity to think about job again. Ultimately users can get good chances to meet with a lot of firms and get a good knowledge for improving further.

For making this application, AppMakr will be mainly used. This software needs almost no programming skills to develop an app which means that it’s really easy to approach and not complicated. This will be mostly used to make a basic frame of application. Since app’s UI can also be customized and previewed without high difficulty. So it is best suited to enterprise application start ups as non-skilled workers are required for the app development. Also SQL server will be used in programming, saving users’ database and easily manage data.

Since this project is worked as a group of people around 3-4 which means that teamwork is very important, communication skills will be very significant in both orally and writing. Also sometimes we need to explain the application to non-technical team members so that it requires communication skills to explain the concepts clearly, easy to understand. Secondly the creativity, as a mobile developer, we need to know how users incorporate their mobile devices into their real lives, and create applications that can support people complete many tasks. Lastly the knowledge of programming language will be necessarily needed for developer. Since a lot of software are used in complexity.

The outcome of this project is to decrease the job seeker who has difficulty to find their appropriate job and get rid of curiosity regarding the job If this project is successfully done, many people will be able to find job much more effectively. They obtain decent information regard their dream job such as what skills they need, what they have to be done so their job satisfaction will ultimately increase. Through this, job selection will be more stabilized and there will be no people who is struggled to find a job which they really want.
